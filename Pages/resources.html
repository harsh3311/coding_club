What is DSA ?
DSA : Data Structures and Algorithms
DSA is your toolbox for tackling some of the toughest challenges in the tech world. It is the first and foremost topic to learn in Computer Science.

Generally, students learn DSA in either of the languages like C++, Java or Python. (if you want to go with Competitive Programming in future then learn C++)
